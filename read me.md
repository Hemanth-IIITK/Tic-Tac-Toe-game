# Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Detects the winner automatically
- Displays a winning message
- Reset button to restart the game
- New Game button to start again
- Simple and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. The game is played on a 3 × 3 grid.
2. Player O and Player X take turns.
3. Click on an empty box to make a move.
4. The first player to get three marks in a row, column, or diagonal wins.
5. Use the **Reset Game** or **New Game** button to play again.
